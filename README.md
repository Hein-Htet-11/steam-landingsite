# Steam Landing Page

A Steam-inspired landing page built with Vue 3 and Vite.

## Features

- Component-based architecture
- Responsive layout
- Hero section
- Game showcase
- Accessories section
- Community section
- Support section

## Tech Stack

- Vue 3
- Vite
- HTML5
- CSS3

## Purpose

This project was created as a frontend practice project while learning Vue.js.
